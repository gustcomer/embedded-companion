================================================================================
Glossary

Jack: a socket designed to receive a jack plug.
Levels of Abstraction: represent the level structure of a typical computer system
  - Level 1: Logic Gate Level -- combinational and sequential circuits.
  - Level 2: Microcode Level
  - Level 3: Instruction Set Architecture Level
  - Level 4: Operating System Level
  - Level 5: Assembly Level
  - Level 6: High-order language level -- languages like C.
  - Level 7: Application Level
Multiplexer: device that selects one of several data inputs to be routed to a single data output.
  - an eight-input multiplexer would have data input lines with inputs from D0 to D7, select control lines with signals S0 to S2 and a single data output line F.
